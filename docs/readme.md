# Vigilane

Vigilane is an innovative application that utilizes computer vision to detect potholes and other hazards on the road in real-time using a live feed from your dashcam.

## Features
- **Real-time Hazard Detection:** Uses advanced computer vision models to identify potholes and road hazards from live dashcam feeds.
- **Cross-Platform Mobile App:** Built with React Native (TSX) for seamless Android and iOS emulation and usage.
- **Robust Backend:** Powered by Python 3.11.9 and Flask for efficient data processing and API management.
- **Cloud Database:** Integrated with Firebase Firestore for reliable and scalable data storage.

## Getting Started

### Prerequisites
- Node.js and npm installed
- Expo Go app installed on your physical device (optional, for testing) or an Android/iOS emulator set up.

### Installation & Running the App

1. **Clone the repository:**
   ```bash
   git clone https://github.com/1Bananas1/hackSLU2026.git
   ```

2. **Navigate to the frontend directory:**
   ```bash
   cd Vigilane
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the Expo development server:**
   ```bash
   npx expo start
   ```

5. **Run the app:**
   - Press `a` to open in the Android emulator.
   - Press `i` to open in the iOS simulator.
   - Or scan the QR code with the Expo Go app on your physical device.
